void setup()
            {
            for (int i = 1; i < 14; i++) {
                pinMode(i, OUTPUT);
            }
            }

            void loop()
            {
            for (int i = 1; i < 16; i++) {
                analogWrite( i - 1 , 200 );
                analogWrite( i , 300 );
                analogWrite( i + 1 , 200 );
                
                delay(200);
            }
            
            for (int i = 1; i < 16; i++) {
                analogWrite( i , 0 );
                
                delay(200);
            }
            }


Требования:

  • 12 виброприводов.
  • 12 n-канальных МОП-транзисторов (полевых транзисторов с МОП-Структурой).

Результат: Моторы вибрируют по циклу.
